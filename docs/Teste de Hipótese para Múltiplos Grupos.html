Teste de Hipótese para Múltiplos Grupos

Análise Paramétrica

ANOVA

A ANOVA é usada para comparar as médias de três ou mais grupos. Ela determina se há pelo menos uma diferença significativa entre as médias de vários grupos.

Exemplo 1
